### Option 2: Using API (PowerShell/curl)
You can use the API directly to create services. Here's a PowerShell example:

```powershell
# First, login to get token
$loginBody = @{
    email = "admin@inhabittech.com"
    password = "12345678"
} | ConvertTo-Json

$loginResponse = Invoke-RestMethod -Uri "http://localhost:5000/api/admin/login" -Method POST -Body $loginBody -ContentType "application/json"
$token = $loginResponse.data.token

# Create Service 1
$service1 = @{
```
