## Usage Instructions

1. **Create Team Members:**
   - Go to Admin Panel → Team → Add New Team Member
   - Fill in the form fields using the mock data above
   - Upload a professional image for each team member
   - Set the `order` field to control display order (lower numbers appear first)
   - Submit the form

2. **Fields Explanation:**
   - **name**: Team member's full name (required)
   - **role**: Job title/role (required)
   - **description**: Professional bio/description (required)
   - **image**: Upload professional headshot (required, recommended: 400x400px)
   - **order**: Display order (number, lower = appears first)
   - **isActive**: Whether team member is active (checkbox)

3. **Image Guidelines:**
   - Use professional headshots
   - Recommended size: 400x400px (square format)
   - File formats: JPG, PNG, WebP
   - Ensure good lighting and professional appearance
   - Maintain consistent style across all team member photos
